# -Desarrollo-de-una-Aplicaci-n-Simple-en-L-nea
El desarrollo de esta aplicación en Java permitió comprender los principios fundamentales de la Programación Orientada a Objetos (POO), como la creación de clases, la instanciación de objetos, y el uso de constructores y métodos. A través del sistema de gestión de estudiantes se evidenció cómo las clases pueden modelar entidades del mundo real.
